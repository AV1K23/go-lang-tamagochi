# 🐶 Консольный Тамагочи на Go

Интерактивная консольная игра-питомец, написанная на языке **Go (Golang)**. Ваш виртуальный друг живет своей жизнью: он голодает, грустит, требует внимания и может даже предупредить об опасности в реальном времени благодаря многопоточности.

## ✨ Особенности

- 🧬 **Объектно-ориентированный подход**: Использование структур и методов для управления состоянием питомца.
- ⚡ **Многопоточность (Concurrency)**: 
  - Отдельная горутина симулирует течение времени (голод растет, настроение падает).
  - Каналы (`chan`) используются для асинхронной передачи тревожных сообщений.
- 🛡️ **Инкапсуляция**: Скрытые поля структуры защищают данные от некорректного изменения извне.
- 🎮 **Интерактивность**: Полноценное консольное меню с выбором действий.
- 🤖 **Умная логика**: 
  - Питомец откажется играть, если слишком голоден.
  - Эмоциональные реакции (смайлики) меняются в зависимости от состояния.
  - Система предупреждений при критическом голоде.

## 🚀 Быстрый старт

### Требования
- Установленный [Go](https://golang.org/dl/) (версия 1.20 или выше).

### Установка и запуск

1. **Склонируйте репозиторий** (или скачайте код):
   ```bash
   git clone <ссылка_на_ваш_репозиторий>
   cd <папка_проекта>
   ```

2. **Инициализируйте модуль** (если еще не сделано):
   ```bash
   go mod init tamagochi
   ```

3. **Запустите игру**:
   ```bash
   go run main.go
   ```

## 🎮 Как играть

После запуска программа предложит ввести имя для вашего питомца. Далее используйте цифровое меню для управления:

| Команда | Действие | Эффект |
| :---: | :--- | :--- |
| **1** | 🍖 **Кормить** | Уменьшает голод на 20, немного повышает настроение. |
| **2** | 🎾 **Играть** | Значительно повышает настроение, но увеличивает голод. *Недоступно, если голод > 70.* |
| **3** | 📊 **Статус** | Показать текущее состояние (голод, настроение, смайлик). |
| **4** | 🚪 **Выйти** | Корректно завершить работу программы. |

### Пример игрового процесса

```text
привет назови своего питомца
> Бобик
Бобик живет своей жизнью...
1. кормить | 2. играть | 3. статус | 4. выйти
> 3
🐶 Бобик 😐 | Голод: 50 | Настроение: 50

> 1
 Бобик поел! Голод: 30, Настроение: 55

> 2
 Бобик поиграл! Настроение: 75, Голод: 40

!!!! спасай Бобик умирает от голода!
```

## 🏗️ Архитектура проекта

Проект демонстрирует ключевые возможности Go:

- **`type Pet struct`**: Основная сущность с приватными полями `hunger` и `mood`.
- **`func (p *Pet) Online(...)`**: Фоновая горутина, которая обновляет состояние каждые 4 секунды.
- **Каналы**:
  - `stopChan`: Для безопасной остановки фоновой горутины при выходе.
  - `alertChan`: Буферизированный канал для передачи срочных уведомлений в главный поток.
